# EnergyWorkshop-Pack

Official resource pack downloads for the EnergyWorkshop Minecraft server plugin.

## Current Version

- Minecraft: 26.1.2
- EnergyWorkshop: 0.15.1
- Resource pack: `EnergyWorkshop-ResourcePack-0.15.1.zip`
- SHA-1: `6ddd83eafa2ac7ded2a0893a6d17904fb03a36b7`

Version 0.15.1 includes custom models for every machine, cable, conveyor, quarry tier, tool, guide, and upgrade card, with improved world brightness.

The Minecraft server downloads this file automatically when a player joins.
