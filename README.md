# EnergyWorkshop-Pack

Official resource pack downloads for the EnergyWorkshop Minecraft server plugin.

## Current Version

- Minecraft: 26.2
- EnergyWorkshop: 0.20.23
- ServerCore: 0.29.0
- Resource pack: `EnergyWorkshop-ResourcePack-0.20.24.zip`
