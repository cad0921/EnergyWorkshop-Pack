# EnergyWorkshop-Pack

Official resource pack downloads for the EnergyWorkshop Minecraft server plugin.

- Minecraft: 26.1.2
- EnergyWorkshop: 0.20.14
- Resource pack: `EnergyWorkshop-ResourcePack-0.20.11.zip`
