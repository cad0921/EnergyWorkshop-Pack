# EnergyWorkshop-Pack

Official resource pack downloads for the EnergyWorkshop Minecraft server plugin.

## Current Version

- Minecraft: 26.1.2
- EnergyWorkshop: 0.20.16
- Resource pack: `EnergyWorkshop-ResourcePack-0.20.15.zip`
- SHA-1: `9651d9e888fe6ef9998abe24da382e06cc526dda`

Version 0.20.15 restyles SpecialCrops fruit and mature plant sprites to fit vanilla Minecraft more closely while retaining all EnergyWorkshop and armor assets.

The Minecraft server downloads this file automatically when a player joins.
