# EnergyWorkshop-Pack

Official resource pack downloads for the EnergyWorkshop Minecraft server plugin.

## Current Version

- Minecraft: 26.2
- EnergyWorkshop: 0.20.33
- ServerCore: 0.31.0
- Resource pack: `EnergyWorkshop-ResourcePack-0.20.33.zip`
- SHA-1: `489eab0a6d41eed4f89ee23c33f33150a6d5d2cf`

Version 0.20.33 rebuilds the original five ground crops as distinct 64x64 Minecraft-style sprites with species-specific mature heights.

The Minecraft server downloads this file automatically when a player joins.
