# EnergyWorkshop-Pack

Official resource pack downloads for the EnergyWorkshop Minecraft server plugin.

## Current Version

- Minecraft: 26.1.2
- EnergyWorkshop: 0.15.0
- Resource pack: `EnergyWorkshop-ResourcePack-0.15.0.zip`
- SHA-1: `4a68c360b7e9660eae9f3da0a9bf9966c3398d66`

Version 0.15.0 includes custom models for every machine, cable, conveyor, quarry tier, tool, guide, and upgrade card.

The Minecraft server downloads this file automatically when a player joins.
