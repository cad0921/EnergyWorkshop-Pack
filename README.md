# EnergyWorkshop-Pack

Official resource pack downloads for the EnergyWorkshop Minecraft server plugin.

## Current Version

- Minecraft: 26.2
- EnergyWorkshop: 0.20.32
- ServerCore: 0.31.0
- Resource pack: `EnergyWorkshop-ResourcePack-0.20.32.zip`
- SHA-1: `31607abe13dea41d86e0b12ef18ee15f3e2aff1e`

Version 0.20.32 redesigns every growth stage of the original five ground crops as distinct classic-Minecraft 16px-style plant sprites.

The Minecraft server downloads this file automatically when a player joins.
