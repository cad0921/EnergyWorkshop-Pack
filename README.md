# EnergyWorkshop-Pack

Official resource pack downloads for the EnergyWorkshop Minecraft server plugin.

## Current Version

- Minecraft: 26.1.2
- EnergyWorkshop: 0.20.16
- Resource pack: `EnergyWorkshop-ResourcePack-0.20.14.zip`
- SHA-1: `b1e6f0748408cbe0435669e3fb59363d8a4c2e1c`

Version 0.20.14 adds strawberry, grape, blueberry, watermelon, and lemon SpecialCrops assets while retaining all EnergyWorkshop and armor assets.

The Minecraft server downloads this file automatically when a player joins.
