# EnergyWorkshop-Pack

Official resource pack downloads for the EnergyWorkshop Minecraft server plugin.

- Minecraft: 26.1.2
- EnergyWorkshop: 0.20.16
- Resource pack: `EnergyWorkshop-ResourcePack-0.20.18.zip`
