# EnergyWorkshop-Pack

Official resource pack downloads for the EnergyWorkshop Minecraft server plugin.

## Current Version

- Minecraft: 26.1.2
- EnergyWorkshop: 0.15.2
- Resource pack: `EnergyWorkshop-ResourcePack-0.15.2.zip`
- SHA-1: `963201ed357f8d13155f499f01c3391947ed1c76`

Version 0.15.2 includes custom models for every machine, cable, conveyor, quarry tier, tool, guide, and upgrade card, with a corrected quest terminal shape.

The Minecraft server downloads this file automatically when a player joins.
