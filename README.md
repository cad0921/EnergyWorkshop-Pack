# EnergyWorkshop-Pack

Official resource pack downloads for the EnergyWorkshop Minecraft server plugin.

## Current Version

- Minecraft: 26.1.2
- EnergyWorkshop: 0.17.0
- Resource pack: `EnergyWorkshop-ResourcePack-0.17.0.zip`
- SHA-1: `14342af8b83ae610265afd0c1286f77d1ede951f`

Version 0.17.0 gives the lumber mill a shaped 3D saw table with a motor, feed rails, log, and exposed blade instead of the shared cube casing.

The Minecraft server downloads this file automatically when a player joins.
